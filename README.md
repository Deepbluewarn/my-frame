## My Frame

사진 공유 소셜 플랫폼 My Frame.

[Demo](https://frame.bluewarn.dev)

## 주요 기능

* Auth0 기반 소셜 로그인 (완료)
* 사진 업로드 (완료)
* 최근 업로드 사진 확인 (완료)
* 사진에 좋아요 추가 (완료)
* 팔로우 기능 (완료)
* 사진에 댓글 달기 (완료)
* 사진 삭제 (예정)
* 로그인 / 회원가입 페이지 커스텀 (예정)
* 회원 정보 조회 및 수정 (예정)

## 기술 스택

* Next.js 14
* MongoDB
* S3 Bucket
* Auth0
* mantine

## 프로젝트 빌드

```bash
git clone https://github.com/Deepbluewarn/my-frame.git
cd my-frame
npm i && npm run build && npm run start
```
